## Définition des collections en Java

* Les collections en Java sont des tableaux à taille dynamique (la taille augmente au fur et à mesure) et peuvent stocker des éléments de types différents.
* Dans la doc officielle, les interfaces sont en _italique_, contrairement aux classes.
* _Collection_ est une interface paramétrique.
* L'interface collection est hérité par :
	- List
	- Set
	- Queue
	
## La classe List
* Un objet de type List est une collection mais ses éléments sont idexés (l'index commence par 0).
* La conséquence de l'indexation des éléments de la liste : la liste peut accepter des doublons.
* L'interface _List_ n'est pas instanciable.

# Cours du 15.02.22

Quand un objet est passé en argument au constructeur, il s'agit d'une aggrégation dans le diagramme UML.
Quand un objet est instancié dans le constructeur, il s'agit d'une composition.

Une association dirigée (flèche pleine) correspond au verbe "avoir".

Une utilisation correspond au verbe "utiliser".

# Cours du 14.02.22

## Définition

Les principes :

* Les interfaces sont crées pour résoudre le problème de l'héritage multiple en Java.
* Les interfaces sont définies par le mot-clef "interface".
* Les interfaces contiennent des constantes et des méthodes abstraites seulement.
* On ne peut pas instancier une interface. Il va falloir implémenter ces méthodes dans des classes concrètes.
* La réalisation d'une interface par une classe sa fait grâce au mot-clef "implements".
* L'implémentation d'une interface se fait à travers le mot-clef implements.
* Une interface joue le rôle d'un contrat qu'on impose aux classes qui l'implémentent.
* Une classe peut implémenter plusieurs interfaces.
* Une interface peut hériter de plusieurs interfaces.
* Les interfaces ont été conçues fonctionnellement afin de remplacer les classes abstraites.
* On peut type les objets d'une classe par l'interface qu'elle implémente.

# Les classes abstraites

Pour rendre une classe abstraite, on ajoute le mot-clé abstract devant son nom.

On ne peut pas instancier un objet d'une classe abstraite. Elle va servir comme modèle.

Une classe abstraite oeut définir des attribus, des méthodes concrètes et des méthodes abstraite.

Si une classe contient en méthode abstraite, elle DOIT être abstraite.

Pour que les classes filles soient croncrètes, il faut obligatoirement redéfinir les méthodes abstraites de la classe mère.
